# Die Axis Calculator

Free tool to calculate coin die axis from photos.

## How to use

1. Upload obverse image → mark 2 reference points
2. Upload reverse image → mark the same 2 points (same order)
3. Press Calculate → get die axis in clock position and degrees

## Features

- Works offline
- No installation needed
- Any browser (desktop/mobile)
- 100% free
